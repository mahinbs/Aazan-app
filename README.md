# AAZAAN - Elite Academic Ecosystem 👋

AAZAAN is a premium, multi-role educational platform built with [Expo](https://expo.dev) and [React Native](https://reactnative.dev/). It connects students with elite subject matter experts and leverages smart AI for instant academic clarity.

## 🚀 Key Ecosystems

### 🎓 Student Portal
- **AI Smart Solve**: Text, audio, and visual problem solving.
- **Expert Directory**: Instant access to top-rated educators.
- **Learning Paths**: Structured categories for Math, Science, Coding, and Business.
- **Public Forum**: Collaborative community for academic questions.

### 👨‍🏫 Professor Portal
- **Dashboard**: Real-time performance analytics and earnings.
- **Availability Management**: Live teaching toggles and scheduling.
- **Virtual Classrooms**: Global outreach tools for webinars and classes.
- **Wallet**: Seamless payout management.

## 🛠️ Development & Build

### Installation
```bash
npm install
```

### Start Local Development
```bash
npx expo start
```

### Build Platform-Specific
- **Web Build**: Generates a static export in the `/web` directory.
  ```bash
  npm run build:web
  ```
- **Android**:
  ```bash
  npx expo run:android
  ```

## 🌐 Deployment
This project is optimized for deployment on **Vercel** via the included `vercel.json` and the `npm run build` command.
